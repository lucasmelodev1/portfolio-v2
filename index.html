Projects

Data Structures and Algorithms in C

Techs: C

Data Structures and Algorithms in C is a project that came to my mind while reading Introduction to Algorithms, by Cormen et al. My idea was to create my own implementation of the data structures that I may come across in my career to practice them and share with my university colleagues.

The decision of the language was lead by my desire to learn C and my desire to understand memory management and safety, so while using languages like Javascript I can understand how much memory it leaks, why and how to solve.

Forgit.nvim, a Neovim git abstraction

Techs: Lua, Neovim

Forgit.nvim is one of my favorite projects because it was born as an idea when I was just starting using Neovim. I wanted to make all git operations inside the editor and I searched well and did not find a solution.

It happened that months later I had the courage to do it, even if I didn't know the language and know nothing about Neovim plugin creation. This project was my first attempt to make public clear documentation that anyone can follow.
